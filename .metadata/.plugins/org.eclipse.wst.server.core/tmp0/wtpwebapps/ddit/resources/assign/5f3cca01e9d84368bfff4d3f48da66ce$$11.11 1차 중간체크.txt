1.회원가입
- 사진업로드
- 아이디 중복확인
- 비밀번호 암호화 하여 저장
- 이메일 중복확인
- 우편번호찾기 api
- 가입 후 이메일 인증후 사용
- 인증메일 클릭시 인증완료 계정 사용가능
-  학생테이블에 같은 데이터 삽입

2. 아이디 비번찾기 
- 아이디 : 이름, 이메일로 아이디 조회
- 비밀번호 : 아이디, 이메일로 비밀번호 찾기 - 입력한 이메일로 암호 변경 메일전송 - 변경링크 1회 사용후 사용불가

3. 로그인
- 아이디 비밀번호 불일치시 경고출력

4. 회원관리
  - 회원 등록
    아이디, 이메일 중복확인, 우편번호 찾기, 회원상태 설정, 회원구분설정(수강생,강사) 후 등록
    회원 구분에 따라서 수강생,강사 테이블에 회원과 같은 데이터 추가
  - 회원정보 수정
    회원정보 모든정보 수정(사진포함), 회원상태 탈퇴로 변경시 탈퇴
5. 마이페이지 
  - 회원정보 수정
  - 이메일 변경시 다시 이메일 중복확인
  - 탈퇴 기능
6. 오른쪽 상단의 로그인 정보표시 - 이름, 미니프로필사진,이메일, 로그아웃 등 
7. 메뉴관리(DB데이터 사용)
  - 왼쪽의 사이드 메뉴 로그인 사용자의 권한별로 메뉴표시
  - 왼쪽 사이드 메뉴 링크 연결
  - 왼쪽 사이드 메뉴 드롭다운 활성화
8. 시험문제관리
  - 시험문제출제
     문제를 유형에 (객관식, 주관식, 서술형) 맞춰서 출제 
     문제 추가,삭제(append, empty) 
     문제안의 객관식 보기항목 추가/삭제 
     저장시 DB에 문제,문제항목으로 나누어 저장
   - 시험문제 조회
     DB에 저장한 문제, 문제항목을 불러와서 JSP에서 append로 출력
   - 시험문제 수정
      문제 수정, 
      문항 수정 중
   - 시험 응시 화면 수정 중
